# Constitución del Proyecto: Stack Tecnológico

Este documento define la verdad oficial sobre el stack de tecnologías y librerías autorizadas para este proyecto. Cualquier cambio en estas tecnologías requiere una actualización previa de este documento y aprobación del usuario.

## Framework y Compilador

- **React v19.2**: Utilización estricta de componentes funcionales y hooks modernos.
- **Vite v7.3**: Herramienta de compilación rápida y servidor de desarrollo.
- **TypeScript v5.9**: Configuración estricta de compilación (`tsconfig.json`).

## Estilos y Apariencia

- **Tailwind CSS v3.4**: Framework de utilidades de diseño móvil-first.
- **Framer Motion v12.4**: Sistema de animaciones fluidas y micro-interacciones.
- **lucide-react**: Set de iconos unificado.

## Gráficos e Interactividad

- **Three.js & @react-three/fiber & @react-three/drei**: Renderizado 3D optimizado (usado para el fondo interactivo).
- **ReactFlow v11**: Librería para renderizar el diagrama de flujo de mapa de habilidades interactivo.

## Validaciones y Formularios

- **React Hook Form v7.76**: Gestión eficiente de formularios.
- **Zod v4.4**: Validación declarativa de esquemas de datos.

## Notificaciones y Feedback

- **Sonner v2.0**: Toast de notificaciones de alto rendimiento.

## Testing y Calidad

- **Vitest v3.2**: Suite de ejecución de pruebas rápidas.
- **React Testing Library v16 + jsdom**: Simulación del DOM en el navegador para testeo de componentes.
- **ESLint v9.39**: Linter de código.
- **Prettier v3.8**: Formateador de código.
