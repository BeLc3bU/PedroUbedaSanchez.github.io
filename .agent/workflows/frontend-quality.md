description: Asegurar estándares de calidad en React y Frontend
---

Guía para mantener el código limpio, performante y escalable en este portfolio de React.

1. **Patrones de Componentes**
   - Separación de lógica y presentación.
   - Uso correcto de hooks (`useEffect` con limpieza).
   - Tipado estricto en Props (TypeScript).

2. **Optimización con Tailwind**
   - Uso de clases dinámicas para temas (`dark:`).
   - Evitar duplicación de estilos mediante componentes o utilidades de Tailwind.
   - Mantener la cohesión visual del diseño "Dark Tech".

3. **Rendimiento**
   - Lazy loading de imágenes.
   - Minimizar renderizados innecesarios.
   - Carga crítica de fuentes e iconos (Lucide React).

// turbo
4. Ejecutar Lint para validar calidad del código
```bash
npm run lint
```
